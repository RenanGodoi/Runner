# Sistema Runner

Neste repositório será usado para o trabalho prático da disciplina de Implementação e Integração de Software (2026). O trabalho visa proporcionar aos estudantes a oportunidade de prática de construção de software por meio do desenvolvimento do Sistema Runner útil para integradores da Plataforma HubSaúde.

Em tempo, este trabalho prático proposto é de interesse real da Secretaria de Estado de Saúde de Goiás (SES) e da Universidade Federal de Goiás (UFG), que realizam um esforço conjunto na definição e implementação da plataforma HubSaúde de interoperabilidade de dados em saúde.

O Sistema Runner tem o objetivo de facilitar o acesso à funcionalidade de execução de aplicações Java via linha de comandos.
